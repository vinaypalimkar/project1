# CDAC Alumni Association Web Application

Welcome to the CDAC Alumni Association Web Application! This application serves as a platform for alumni to connect, share updates, and access resources.

## Features

- **Welcome Note**: Introduction and scrolling images.
- **Jobs & Internships**: List of job opportunities and internships.
- **News and Updates**: Latest news and announcements.
- **Distinguished Alumni**: Testimonials and videos of distinguished alumni.
- **Latest Members**: Profiles of recently joined members.
- **Sidebar**: Connect with classmates via social media.

## Technologies Used

- Frontend: React, Redux, Material-UI
- Backend: Node.js, Express.js
- Deployment: Vercel, Render
- Other: Bootstrap Icons, Figma for design

## Getting Started

### Prerequisites

- Node.js (version >= 12.0.0)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/cdac-alumni.git
   cd cdac-alumni
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Open your browser and navigate to `http://localhost:3000` to view the application.

### Deployment

- The application is automatically deployed using Vercel for the frontend and Render for the backend.
